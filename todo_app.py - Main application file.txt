# Todo List Application

A simple Todo List Application built using Python Tkinter and CSV file handling.

## Features

* Add Tasks
* Edit Tasks
* Delete Tasks
* Search Tasks
* Show All Tasks
* Clear All Tasks
* Task Counter
* Save Tasks to CSV File
* Load Tasks Automatically
* Confirmation Dialog Before Clearing All Tasks

## Technologies Used

* Python
* Tkinter
* CSV Module

## Project Structure

todo_app.py - Main application source code

task_list.csv - Stores task data

README.md - Project documentation

## How to Run

1. Install Python.
2. Download or clone the repository.
3. Open terminal in the project folder.
4. Run:

python todo_app.py
