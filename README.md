# To-Do List Application

## Overview

To-Do List Application is a Python-based desktop application developed using the Tkinter library. The application helps users organize and manage their daily tasks efficiently by allowing them to add, edit, search, delete, save, and load tasks.

## Objectives

* To develop a user-friendly task management system.
* To apply Python programming concepts such as lists, functions, file handling, and exception handling.
* To improve productivity through effective task organization.

## Features

* Add new tasks.
* Edit existing tasks.
* Delete selected tasks.
* Search tasks by keyword.
* View all saved tasks.
* Clear all tasks with confirmation.
* Prevent duplicate task entries.
* Display the total number of tasks.
* Save tasks to a CSV file.
* Load previously saved tasks automatically when the application starts.

## Technologies Used

* Python
* Tkinter (Desktop GUI Development)
* CSV Module (File Handling)

## Project Structure

* `todo_app.py` – Main source code
* `task_list.csv` – Stores task records
* `README.md` – Project documentation

## How to Execute

1. Open the project folder in Visual Studio Code.
2. Run the Python program using:

```bash
python todo_app.py
```

3. Use the graphical user interface to manage tasks.

## Learning Outcomes

Through this project, the following concepts were implemented:

* Python Lists
* Functions
* Conditional Statements
* Loops
* Exception Handling

This project demonstrates the practical application of Python programming concepts in developing a simple and effective task management system.
